Web-Based Browser Tool

Educational Project Documentation

Project Overview

This tool is a web-based browser application that allows browsing the internet through a simple iframe interface. It includes privacy-enhancing features and a stealth mode to prevent monitoring of your browsing activity.

Educational Purposes Only: This tool is for educational purposes and to demonstrate web privacy concepts. Using this tool to circumvent monitoring may violate your school's acceptable use policies.

How to Use

  1. Basic Version (index.html):
    • Simple interface with stealth mode toggle
    • Enter URLs in the address bar and click GO
    • When GoGuardian monitoring is detected, the screen will temporarily appear black to them
    • Use the Bypass Mode toggle to keep browsing even when monitoring is detected
  2. Advanced Version (advanced.html):
    • More sophisticated monitoring detection
    • Disguised as a document viewer
    • Dark theme option
    • Bypass Mode toggle to continue browsing when GoGuardian is active
    • Secret key combination: Press 'z' three times quickly to manually toggle stealth mode
  3. Terminal Version (terminal.html):
    • Command-line interface that appears as a legitimate terminal
    • Type 'help' to see available commands
    • Use 'browse [url]' to navigate to websites
    • Type 'bypass' to toggle the bypass mode on/off
    • Press Esc three times quickly as an emergency shortcut
    • Command history with arrow keys
    • Tab completion for commands
Tip: The terminal version provides the best camouflage in school environments, as it looks like a legitimate programming tool. With Bypass Mode enabled, you can continue browsing even if GoGuardian is actively monitoring.

Features

  • Stealth Mode: Hides your browsing when monitoring is detected
  • Bypass Mode: Allows browsing even when GoGuardian is active (new feature)
  • Monitoring Detection: Uses various techniques to detect when your screen is being monitored
  • Deception Layer: Shows an innocent "Document Viewer" or "Terminal" screen to monitoring software
  • Domain Blocking: Prevents connections to monitoring domains
  • Multiple Interfaces: Choose between standard, document viewer, or terminal interfaces

Terminal Commands

The terminal version offers a realistic command-line experience with the following commands:

  • help: Display list of available commands
  • browse [url]: Open browser mode with optional URL (e.g., browse google.com)
  • status: Display current protection status
  • stealth: Toggle stealth mode on/off
  • clear/cls: Clear terminal output
  • emergency: Immediately block screen
  • about: Display information about the tool
  • Ctrl+C: Cancel current command or exit browser mode

Technical Details

The tool works by containing your browsing within an iframe, which makes it difficult for monitoring software to see what you're browsing. Additionally, it employs several techniques to detect monitoring activity:

  • Focus/blur detection to identify screen capture attempts
  • Mutation observers to detect monitoring scripts
  • Mouse movement analysis to identify automated control
  • DOM scanning for monitoring-related elements

When monitoring is detected, the tool displays a deception layer that appears to be an innocent application.
