Introduction

Blackout is a browser-based protection system designed to provide distraction-free browsing while protecting your privacy from monitoring software such as GoGuardian. The 2.0.0 update brings multiple interface themes, enhanced detection capabilities, and a more robust protection system.

New Features in 2.0.0

  • Multiple Interface Themes - Choose between Terminal, Document, and Editor themes
  • GoGuardian Dependency - Requires GoGuardian to be installed for full functionality (or use secret code)
  • Improved Mouse Movement Detection - Fewer false positives in monitoring detection
  • Enhanced "secret" Command - Added more functionality and developer options
  • Fixed Ctrl+C Functionality - Now properly exits browser mode in terminal
  • Integrated Unblocker Services - Access restricted sites using built-in proxy options
  • More Realistic Interfaces - Redesigned to look like standard applications

Themes

Blackout 2.0.0 includes three main themes that can be switched between:

Terminal Theme

A command-line interface with advanced features:

  • Command-driven navigation
  • Integrated browser capability
  • Advanced monitoring detection
  • Secret commands for additional functionality

Document Theme

Appears as a standard document viewer:

  • Clean, professional interface
  • Simulated document editing environment
  • Side navigation with document folders
  • Looks like a legitimate work environment

Editor Theme

A code editor interface for technical environments:

  • Code syntax highlighting
  • File explorer panel
  • Tab-based file navigation
  • Status bar with developer information

GoGuardian Dependency

Blackout 2.0.0 implements a security feature that requires the GoGuardian extension to be installed and active for full functionality. This helps ensure the tool is only used in environments where protection is actually needed.

Bypass with Secret Code

If you need to use Blackout without GoGuardian installed (for testing or development), you can use the secret key sequence "lights-out" to bypass this requirement.

Bypass Mode

Bypass Mode allows content to remain visible even when monitoring is detected. This is useful when:

  • You need to view content without interruption
  • You're experiencing false positive detections
  • You're in a controlled environment where monitoring is expected

To toggle Bypass Mode:

  • In Terminal theme: Use the bypass command
  • In Document/Editor themes: Use the settings menu
ℹ️ Note

Bypass Mode is enabled by default in this version. When enabled, monitoring software will see your active browser window, but you'll still receive notifications when monitoring is detected.

Terminal Commands

The Terminal theme provides the following commands:

Command Description
help Show available commands
browse [url] Open browser with optional URL
status Display current protection status
stealth Toggle stealth mode on/off
bypass Toggle monitoring bypass mode
check Check for GoGuardian extension
clear Clear terminal output
emergency Immediately block screen
update Show update information
about Show information about this tool
secret Access secret developer options

Secret Commands

The secret command provides additional advanced functionality:

Command Description
secret help Show secret commands
secret unlock Force unlock all features
secret block Simulate monitoring detection
secret reset Reset all settings
secret bypass Toggle bypass mode
secret monitor Show detected monitoring elements
secret theme Change terminal appearance

Known Limitations

Please be aware of the following limitations:

  • Browser-based protection has inherent limitations and cannot completely prevent all forms of monitoring
  • Some features may not work consistently across all browsers and devices
  • Aggressive monitoring software may detect and block Blackout
  • Detection is based on patterns and may occasionally produce false positives or negatives
